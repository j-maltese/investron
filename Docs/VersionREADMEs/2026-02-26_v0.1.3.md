# Investron v0.1.3 — Value Screener with S&P 500 Scanner

**Date:** 2026-02-26

## Overview

Release 0.1.3 includes 3 new features.

## New Features

- Value Screener with background S&P 500 scanner — scans all tickers and scores them on value investing metrics
- Composite scoring with weighted factors: Margin of Safety, P/E, FCF Yield, Earnings Yield, P/B, ROE, D/E, Dividend Yield
- Dockerfile and release notes integration for automated CI/CD
