# Investron v0.1.2 — Watchlist and Deployment

**Date:** 2026-02-25

## Overview

Release 0.1.2 includes 4 new features, 4 enhancements, 4 bug fixes.

---

## New Features

- Full-stack investing research application with watchlist management, stock detail views, and financial metrics
- In-app documentation page with project overview and architecture details
- Railway backend deployment with Procfile support
- Vercel frontend deployment

---

## Enhancements

- Concurrent watchlist price fetching — prices for all tickers load in parallel instead of sequentially
- yfinance calls offloaded to thread pool to avoid blocking the async event loop
- Rate limiter tuned to allow small burst (5 requests / 2 seconds) for concurrent watchlist loads
- Full-width layout for header and main content area

---

## Bug Fixes

- Fixed yfinance failing on cloud servers by adding curl_cffi with browser impersonation
- Fixed CORS_ORIGINS environment variable parsing for comma-separated origins
- Fixed TypeScript build errors for Vercel deployment
- Upgraded yfinance for curl_cffi compatibility
