# Investron v0.1.24 — Wheel Screener Candidates and Filing Refresh

**Date:** 2026-03-05

## Overview

Release 0.1.24 includes 4 new features, 3 enhancements.

---

## New Features

- Wheel strategy now uses screener-driven dynamic candidate selection instead of hardcoded ticker list
- Sector diversification constraint (max 2 positions per sector) for Wheel candidates
- Refresh Filings button on Filings tab — re-fetch filing list from EDGAR without triggering RAG indexing
- Index for AI button on Filings tab — trigger RAG embedding pipeline on demand

---

## Enhancements

- Wheel strategy capital bumped from $5,000 to $30,000 (paper money) for broader position coverage
- Strategy card shows screener mode description when screener_enabled, falls back to fixed ticker list for legacy mode
- Open-position tickers are always included in Wheel candidates to prevent position orphaning
